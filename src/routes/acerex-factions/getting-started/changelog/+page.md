Below you will find a comprehensive list of all changes made to Acerex Factions. This includes updates to gameplay, plugins, balancing, bug fixes, and server infrastructure. Entries are listed in reverse chronological order, with the most recent update at the top.

## How to Read Changelogs

Each changelog entry follows a consistent format to make it easy to understand what changed and why. Here's what each tag means:

| Tag | Meaning |
|---|---|
| ✅ Added | New features or content introduced |
| 🔧 Changed | Adjustments to existing features or balancing |
| 🐛 Fixed | Bug fixes and patches |
| ❌ Removed | Features or content taken out |
| ⚠️ Known Issues | Active bugs or issues being investigated |

## Version 1.0.0 - July 1st 2020

:::note
This is just a temp changelog to see how they will look.
:::

**Summary:** Mid-season balance patch focused on economy adjustments and 
PvP tuning, alongside several quality of life improvements.

**✅ Added**
- Added /f top to display the top 10 factions by power
- New crop: Sugarcane added to the shop with updated sell prices
- Added a 5-minute grace period for new players before PvP is enabled

**🔧 Changed**
- Reduced TNT cannon cooldown from 10s to 7s
- Adjusted shop prices across all ore tiers — see Economy Guide for 
  updated values
- Faction power loss on death reduced from 3 to 2 for members below 
  rank Lieutenant

**🐛 Fixed**
- Fixed an issue where faction claims near world border were not 
  registering correctly
- Fixed /f home occasionally teleporting players underground
- Resolved a dupe exploit involving hoppers and chest minecarts

**❌ Removed**
- Removed the Starter Kit claim cooldown reset on relog

**⚠️ Known Issues**
- Scoreboard occasionally flickers on respawn — under investigation
