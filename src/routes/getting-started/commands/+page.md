# Player Commands

## Faction Commands

| Command | Description |
|---|---|
| `/f create <name>` | Create a faction |
| `/f disband` | Disband your faction |
| `/f invite <player>` | Invite a player |
| `/f join <faction>` | Join a faction (requires invite) |
| `/f leave` | Leave your faction |
| `/f kick <player>` | Kick a member |
| `/f promote <player>` | Promote a member |
| `/f demote <player>` | Demote a member |
| `/f claim` | Toggle claim mode |
| `/f unclaim` | Unclaim the region you're in |
| `/f sethome` | Set faction home (in own claim) |
| `/f home` | Teleport to faction home |
| `/f chat` | Toggle faction chat |
| `/f map` | Show a map of nearby claims |
| `/f info [faction]` | View faction info |
| `/f who` | View faction info |
| `/f list` | List all factions |
| `/f balance [faction]` | Check faction vault balance |
| `/f bal [faction]` | Check faction vault balance |
| `/f deposit <amount>` | Deposit money into the faction vault |
| `/f withdraw <amount>` | Withdraw money from the faction vault (Captain+) |

## Economy / Player Commands

| Command | Description |
|---|---|
| `/lives` | Show your lives |
| `/lives check [player]` | Check lives for yourself or another player |
| `/lives revive <player>` | Revive a deathbanned player |
| `/pvptimer` | Disable your PvP protection early |
